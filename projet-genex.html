Projet GenEx

Générateur & Correcteur d'Exercices pédagogiques automatisé basé sur l'IA Google Gemini

Python 3 Flask Gemini API Bootstrap 5 Render
🌐 Visiter le site en ligne

Aperçu de l'Application

Module Générateur - Saisie

Interface de configuration (Matière, Niveau, Précisions) pour créer un sujet

Module Générateur - Résultat

Affichage du sujet formaté prêt pour l'impression ou le téléchargement Word

Module Correcteur - Upload

Interface d'envoi pour soumettre un fichier d'exercices au format .docx

Module Correcteur - Évaluation

Affichage du correctif détaillé, de la note sur 20 et des axes d'amélioration
